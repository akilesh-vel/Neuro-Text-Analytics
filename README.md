🧠 NeuroText Analytics – Client-Side Text Analysis Dashboard

## 📌 Overview

NeuroText Analytics is a responsive client-side web application developed using **HTML5, CSS3, and Vanilla JavaScript**. The application analyzes user-entered text in real time and displays useful text statistics through an interactive dashboard.

This project was built to strengthen frontend development skills and demonstrate JavaScript DOM manipulation, event handling, and responsive web design.



## 🚀 Features

- Character Count
- Word Count
- Sentence Count
- Paragraph Count
- Reading Time Estimation
- Speaking Time Estimation
- Longest Word Detection
- Average Word Length
- Number Count
- Special Character Count
- Keyword-Based Sentiment Analysis
- Sentiment Score
- Copy Text
- Download Report
- Remove Extra Spaces
- Clear Text
- Responsive Dashboard

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser or use **Live Server** in Visual Studio Code.
4. Enter text into the text area.
5. View real-time analysis and use the available text utility features.
